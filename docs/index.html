强大的功能特性

一切为了更好的终端体验

多会话管理

支持同时管理多个 SSH 会话，智能分组，快速切换，提升工作效率

快捷命令

自定义快捷命令，一键执行常用操作，支持变量替换和命令片段

智能补全

命令自动补全、路径补全、历史命令补全，让输入更快更准确

SFTP 文件传输

内置 SFTP 客户端，支持文件上传下载，拖拽操作，传输队列管理

安全可靠

支持密码、密钥认证，会话锁定，密码加密存储，保护您的数据安全

端口转发

支持本地和远程端口转发，轻松访问内网服务，满足各种网络需求

服务器监控

实时监控服务器状态，CPU、内存、磁盘使用情况一目了然

主题定制

多种精美主题，支持自定义配色，打造专属终端风格

命令历史

完整的命令历史记录，支持搜索过滤，快速找到历史命令

会话模板

创建会话模板，快速部署相似配置，批量管理服务器

审计日志

详细的操作日志记录，支持导出，满足合规审计需求
